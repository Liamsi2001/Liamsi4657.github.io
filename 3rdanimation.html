This algorithm illustrates the shortest path finding problem for tethered point robot in the retracting model.

Instructions:

  1. Create obstacles by clicking the 'New obstacle' button and adding points.
  2. Set the start and end points using the 'START' and 'END' buttons.
  3. Set the anchor point by clicking the 'ANCHOR' button and adding a point.
  4. Create the tether by placing points and clicking the 'TETHER' button.
